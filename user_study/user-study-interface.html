Image Classification User Study

In this study, you will learn to classify images into two categories - 0 and 1.

This study will show you several sets of images:

  1. 100 reference images with their categories (1 minute)
  2. Images flipping from Category 0 to Category 1 (1 minute)
  3. Images flipping from Category 1 to Category 0 (1 minute)
  4. 50 test images one by one (you'll classify each as Category 0 or 1)

When you're ready to begin, click the Start button.
